## 1. Общее описание

**AkitaStudio** — веб-приложение детейлинг-студии. Через сайт вы можете:

- ознакомиться с перечнем услуг и их стоимостью;
- записаться на обслуживание автомобиля онлайн;
- отслеживать статус и прогресс выполнения своего заказа;
- просматривать фотоотчёты по завершённым работам;
- управлять личными данными в личном кабинете.

Сайт работает в любом современном браузере (Chrome, Firefox, Safari, Edge). Мобильная версия поддерживается в полном объёме.

## 2. Главная страница

При открытии сайта отображается главная страница с блоками:

- **Приветственный баннер** — краткое описание студии и кнопка «Записаться».
- **Популярные услуги** — карточки основных направлений (полировка, химчистка, детейлинг, бронирование).
- **Преимущества студии** — почему стоит выбрать AkitaStudio.
- **Портфолио** — превью последних работ.
- **Навигация** — в шапке находятся ссылки на все разделы сайта.

## 3. Услуги

Раздел **«Услуги»** → `/services`

Здесь представлен полный список услуг студии, сгруппированный по категориям:

| Категория | Примеры услуг |
|---|---|
| Полировка | Полировка кузова (1 ступень), Полировка кузова (2 ступени) |
| Химчистка | Химчистка салона (базовая), Химчистка потолка |
| Детейлинг | Детейлинг двигателя |
| Бронирование | Антигравийная плёнка |

Каждая карточка услуги содержит:
- Название и описание
- Базовую стоимость (₽)
- Ориентировочную продолжительность

Кнопка **«Записаться»** на карточке услуги переводит сразу на форму записи с предвыбранной услугой.

## 5. Запись на услугу

Раздел **«Запись»** → `/booking`

### 5.1 Шаг 1 — Выбор услуг

1. Нажмите на вкладку категории (Полировка, Химчистка и т. д.).
2. Отметьте одну или несколько услуг. Можно выбрать услуги из разных категорий.
3. Справа отображается итоговая стоимость по выбранным услугам.

### 5.2 Шаг 2 — Заполнение формы

| Поле | Обязательное | Описание |
|---|---|---|
| Имя | Да | Ваше имя для связи |
| Телефон | Да | Формат: +7 (___) ___-__-__ |
| Марка автомобиля | Да | Начните вводить — появятся подсказки |
| Модель автомобиля | Да | Выберите из списка или введите вручную |
| Желаемая дата | Нет | Предпочтительная дата визита |
| Желаемое время | Нет | Предпочтительное время (10:00–20:00) |
| Комментарий | Нет | Дополнительные пожелания |

> **Примечание.** Если вы вошли в личный кабинет, ваши данные могут быть использованы автоматически.

### 5.3 Шаг 3 — Отправка заявки

1. Поставьте галочку «Я согласен с условиями».
2. Нажмите **«Записаться»**.
3. После успешной отправки на экране появится сообщение подтверждения. Менеджер свяжется с вами для согласования точного времени.

> **Совет.** Чтобы заявка попала в ваш личный кабинет автоматически — **войдите в аккаунт** перед оформлением записи.

## 6. Регистрация и вход

### 6.1 Регистрация

Страница **«Регистрация»** → `/register`

1. Заполните поля: Имя, Фамилия, Телефон, Логин, Пароль (не менее 6 символов).
2. Нажмите **«Зарегистрироваться»**.
3. После регистрации вы автоматически перейдёте на страницу входа.

Если телефон, указанный при регистрации, совпадает с телефоном из существующей заявки — ваш аккаунт автоматически свяжется с ней, и история заказов будет видна в личном кабинете.

### 6.2 Вход

Страница **«Вход»** → `/login`

1. Введите логин и пароль, заданные при регистрации.
2. Нажмите **«Войти»**.
3. После входа вы будете перенаправлены в личный кабинет.

### 6.3 Выход

Нажмите на иконку профиля в шапке → **«Выйти»**.

## 7. Личный кабинет

Раздел → `/profile` (доступен только после входа)

Личный кабинет состоит из двух вкладок.

### 7.1 Вкладка «Мои заказы»

Отображается список всех ваших заявок, отсортированных от новых к старым.

#### Карточка заказа содержит:

| Элемент | Описание |
|---|---|
| Статус | Цветной бейдж с текущим статусом заказа |
| Автомобиль | Марка и модель |
| Услуги | Перечень заказанных работ |
| Прогресс | Полоса прогресса выполнения (при наличии) |
| Фотоотчёт | Миниатюры фото (нажмите, чтобы открыть галерею) |
| Даты | Дата подачи заявки и желаемая дата визита |
| Сумма | Стоимость заказа |

#### Статусы заказа:

| Статус | Значение |
|---|---|
| 🟡 Новый | Заявка принята, ожидает подтверждения менеджером |
| 🟠 В работе | Автомобиль принят, работы ведутся |
| 🔵 Готово | Работы завершены, автомобиль готов к выдаче |
| 🟢 Выдан | Автомобиль получен клиентом |
| 🔴 Отменён | Заказ отменён |

#### Действия с заказом:

**Отменить заявку** — доступно для заказов в статусах «Новый» и «В работе», если до желаемой даты остаётся более 1 часа.

**Перенести заявку** — доступно для заказов в статусах «Новый» и «В работе». При переносе:
1. Нажмите **«Перенести»**.
2. Укажите новую дату (не ранее сегодняшнего дня) и время.
3. Нажмите **«Подтвердить»**.

> Перенос недоступен для завершённых и отменённых заказов.

### 7.2 Вкладка «Мои данные»

Раздел для просмотра и редактирования личной информации.

| Поле | Редактируемое |
|---|---|
| Имя | Да |
| Фамилия | Да |
| Отчество | Да |
| Телефон | Нет (для изменения обратитесь к менеджеру) |
| Email | Да |

Нажмите **«Сохранить изменения»** для применения правок.

## 8. О студии и контакты

- **«О нас»** → `/about` — история студии, команда, сертификаты.
- **«Контакты»** → `/contacts` — адрес, телефон, режим работы, форма обратной связи.

### Форма обратной связи

На странице «Контакты» расположена форма для отправки вопроса или пожелания:

1. Укажите имя, телефон или email, сообщение.
2. Нажмите **«Отправить»**.
3. Менеджер ответит вам в ближайшее рабочее время.

## Часто задаваемые вопросы

**Я подал заявку, но не вижу её в личном кабинете.**  
Заявка появляется автоматически, если вы были авторизованы в момент записи и номер телефона в форме совпадает с номером, указанным при регистрации. Если заявка не появилась — обратитесь к менеджеру по телефону.

**Могу ли я записаться без регистрации?**  
Да. Заявка будет обработана менеджером в обычном порядке. Для отслеживания статуса онлайн необходимо зарегистрироваться.

**Как узнать, что мой автомобиль готов?**  
Статус заказа обновляется в личном кабинете. Также менеджер свяжется с вами по телефону.

**Как изменить телефон?**  
Номер телефона является идентификатором аккаунта и не может быть изменён через личный кабинет. Обратитесь к администратору студии.
